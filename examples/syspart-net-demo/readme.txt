Данное приложение демонстрирует работу сети.
JetOS по сети начинает общаться с скриптом, который выполняется
на инструментальной машине. Для запуска необходимо:

1. Выставить destination mac адрес.
    В случае запуска на qemu с tap устройства это mac адрес tap устройства.
    В случае запуска на железе это mac адрес инструментальной машины.

    Для этого неоходимо в файле 'P2/components-glue/config.yaml'
    у 'udp_ip_sender_1' поменять 'dst_mac'. После необходимо выполнить
    'scons cdeveloper=1'
2. запустить jetos
    В случае запуска на qemu выпонить 'scons run-tap'

3. запустить
    misc/net_demo.py
    В нем должна отобразиться скорость сети
